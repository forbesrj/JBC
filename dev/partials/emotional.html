Emotional Experience

Just Better Code develops while constantly considering the experience of the person who will be using the site. A website should not just be a path from person to information, the path should be effortless and enjoyable.

A website can be positively emotionally impactful in two ways:

  • It is so easy to navigate that you do not have time to generate a negative opinion about the site.
  • It connects with you in an unexpected way.
